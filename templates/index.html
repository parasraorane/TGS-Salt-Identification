TGS Salt Identification Challenge

Segment salt deposits beneath the Earth's surface. Join now and compete!

This app takes seismic images for the competition "Tgs Salt Identification Challenge" in ".png" format, processes it and returns predictions for segmented salt depost maps using a pre-trained tensorflow U-NET model which achieved an iou score of 0.786 on the main competition. Below are the requirements the input data must satisy, which if aren't, will give errors.

  • File should be a png image stored in ".png" format
  • Image should have standard height and width of 101 pixels
  • Model can process only one image at a time
Output will be the predicted mask stored in png format of the same width as the image. Download will automatically start after the file is uploaded,processed and the output is generated.

Warning: App can process only one image at a time.
